# Microsoft DP-100 Reference Notebooks 

This repository contains reference notebooks for the Azure ML Python SDK and Databricks courses, which were used to prepare for the Microsoft DP-100 Data Scientist exam.

Descriptions for the two folders contained in this repository are given below.

## Azure ML Python SDK Notebooks

This folder contains completed notebooks, with outputs, from the Microsoft repository located at:

https://github.com/MicrosoftLearning/mslearn-dp100

## Databricks Notebooks

This folder contains completed notebooks from the Microsoft Learn Databricks course, located at:

https://docs.microsoft.com/en-us/learn/paths/perform-data-science-azure-databricks/

The notebooks were ran in the Databricks environment, and then saved as HTML files to preserve formatting.
